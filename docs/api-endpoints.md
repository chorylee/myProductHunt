# API Endpoints

## HTML API

### Root
  - `GET /` -loads React web app

## JSON API

### Users
  - `GET /api/users`
  - `GET /api/users/:id`
  - `POST /api/users`

### Session
  - `GET /api/session`
  - `POST /api/session`
  - `DELETE /api/session`

### Products
  - `GET /api/products`
  - `POST /api/products`
  - `GET /api/products/:id`
  - `PATCH /api/products/:id`

### Discussion
  - `GET /api/discussion`

### Comments
  - `POST api/comment`
